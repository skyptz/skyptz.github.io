多说即将关闭

发表于 2017-03-27   |   分类于 闲时博客   |  

非常不好的一个事情就是，一直在加班加点忙公司官网，然后突然得知，多说关闭了，于是登上我的博客和后台看了看，确实是这样。
为此很难过。
大概不植入广告不加宣传不能变现的评论插件，基本上都无可避免会面对这样的结果。
我不算使用多说的一个老用户，甚至说应该是很新，也不是一个很好的专情的用户，毕竟我会吐槽多说抽了我的评论，不够稳定不够那么有吸引力。
作为一个用户，我是矫情并且挑剔的。
但是，我承认我是被这样一款称得上良心的产品打动了，或者这就说俗话说的爱之深，责之切？
不得不说这个插件非常简捷，使用过程中废话不多。
很多的评论插件都会给你各种限制，要么是植入广告，要么是转移流量，最后归到底都是钱。
也因为如此，即使经历那么多风雨的多说也要关闭了。
有人可能怪竞争，怪社会，怪各种原因。
但是，它确确实实走了。
以后，该何去何从。
