关于用Hexo搭建github静态博客

发表于 2016-09-20   |   分类于 基本教程   |  

今天来讲讲我是怎么用Hexo 3.0以上版本【本机版本3.2.2】在wondows上搭建的github静态博客。
3.0以下版本有可能出现教程不适用的情况。
另外，关于这方面百度上的教程大多不完全可行，教程是自己慢慢摸索得来，仅供参考。

建立github仓库

github可以进行 Git 代码仓库托管，首先要有一个仓库；

  • 申请一个github账号，登录之后，点击new respository
    在出现的界面中，Repository name必须是两种格式中一种：github用户名.github.io 或 github用户名.github.com
    点击create respository新建，这就是你的博客在github上的基地了；

注意，在git中如果使用SSH地址访问github，如（（git@github.com:github用户名/github用户名.github.io.git），必须要在这里配置一下SSH-key，当然如果使用HTTPS协议，可以忽略这步操作；

部署到Github

  • 用记事本打开，找到下面文字，一般在文本末尾：

  • 修改为：
    deploy:
    type: git
    repo: https://github.com/github用户名/github用户名.github.io.git
    branch: master
    保存就可以了；
